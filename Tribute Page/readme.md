# Tribute Page

This technical documentation page project was created as one of my responsive web design practices on <a href="https://www.freecodecamp.org/learn/2022/responsive-web-design">freeCodeCamp</a>. It is a tribute page dedicated to honoring the life and contributions of Norman Borlaug, a renowned agronomist and humanitarian. The tribute page contains important timeline of his life, his legacy and the positive impact he made on global agriculture and food security.

## Technologies Used

- HTML
- CSS
